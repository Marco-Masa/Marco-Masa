# 🥇 Calculadora de Consumo Elétrico Inteligente
## 🚀 Objetivo
- Este projeto em Python tem o objetivo de ajudar os usuários a estimar quanto um aparelho gasta de energia elétrica por mês, com base em dados simples de uso. E ainda oferece a opção de calculo do custo mensal estimado.
- O programa solicita o **nome do usuário**, **nome do aparelho**, **potência do aparelho** e **tempo médio de uso**, além de disponibilizar a opção do usuário em informar o **custo por kWh**, caso o usuário não queira utilizar um valor padrão.
## 🔧 Instruções de Uso:
- Informe o nome do aluno.
- Informe o nome do aparelho.
- Digite a potência do aparelho.
- Digite o tempo médio de uso diário do aparelho.
- Caso queira a estimativa de custo mensal, poderá informar o valor do custo por kwh, se não quiser utilizar o valor padrão do programa.
## ✔ Exemplo dos resultados (Usuário exemplo: "José"):
- Obrigado, **José** o consumo estimado da(o) Televisão é de **21.60 kWh/mês**.
- **José**, o custo mensal estimado do consumo da(o) Televisão é de **R$ 16.20**.
