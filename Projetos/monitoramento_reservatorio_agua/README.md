# 🥇 Sistema de Monitoramento dos Níveis de Água de um Reservatório
## 🚀 Objetivo
- Este projeto tem o objetivo de monitorar os níveis de água de um reservatório conforme o valor percentual do nível da água apresentando uma resposta da situação.
- O programa inicialmente realiza uma simulação demonstrando todas as mensagens possiveis em cada passibilidade de nível de água. Em seguida solicita o **valor percentual do nível de água**.
- A pesquisa retorna a situação do nível de água do reservatório classificando em: Muito baixo (crítico), Baixo, Médio, Alto, Muito alto (alerta), com o texto realçado de acordo com a gravidade da situação.
## 🔧 Instruções de Uso:
- Informe o valor percentual do nível de água.

## 🌟Recursos
- A entrada de **Nível de Água** possui validação para números válidos (0 - 100) e informação em brando.
